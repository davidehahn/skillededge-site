Floor SOPs
Standardized procedures for all core warehouse workflows. Every operator follows the same steps, every time.
📥
Inbound Receiving
All shipments
  1. Count cartons against BOL before driver leaves
  2. Photograph any visible damage before signing
  3. Scan each carton into WMS receiving queue
  4. Assign putaway location — high velocity near dock
  5. Update WMS putaway complete within 2 hours of receipt
📤
Pick & Pack
Standard orders
  1. Print wave or pull single order from WMS queue
  2. Pick to tote — verify SKU scan matches order line
  3. Pack per client spec (check Client Playbook first)
  4. Apply correct label — verify ship-to address
  5. Scan carton closed, place on outbound conveyor
↩️
Returns Processing
All accounts
  1. Scan return label — log into WMS returns queue
  2. Open and inspect per client protocol
  3. Grade: Restock / Quarantine / Destroy
  4. Photograph if damaged or opened
  5. Log disposition in WMS and notify account manager
🔄
End of Shift Checklist
All shifts
  1. Clear all open orders from WMS queue or flag exceptions
  2. Restock packing supplies at each station
  3. Return all pick carts to charging stations
  4. Log any unresolved exceptions in Exception Log
  5. Confirm outbound manifests are closed in WMS
