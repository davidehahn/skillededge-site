Pre-Setup Checklist
Verify before making first cut on any new setup.
Program number matches router/traveler
Material cert verified — alloy and temper correct
Tool offsets entered and confirmed
Work offsets set and verified
Fixture clamped — torque spec confirmed
Coolant on, correct concentration
🔍
First Article Inspection
Measure and document before releasing to full production run.
All critical dimensions measured vs. print
Surface finish verified
Thread gauged — go/no-go
First article form signed
Operator name and timestamp logged
🗂️
Post-Run Teardown
Complete after last part of the run is done.
Tool condition logged — replace or note wear
Fixture cleaned and returned to rack
Any deviations noted on traveler
Machine wiped down
Next job setup info confirmed with lead
